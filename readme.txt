El problema está dividido en dos funciones:
-check_palindrome, recibe por parametro un String y retona como respuesta un String
en ingles evaluando concatenado con un Boolean, ejemplo: 'Is palindrome: True'.

-Unit_testing, recibe por parametro el directorio donde se encuentran los archivos txt para las pruebas unitarias,
y file_exp, el cual es la expresión de los archivos. ejemplo: 'unit_test_1.txt'. Esta función crea un archivo de
nombre 'unit_test_*_result.txt' donde * es el numero de la prueba unitaria.
Los documentos 'unit_test_*.txt' contienen un String, el cual es el palindromo a evaluar.
